# PCA Analysis of Gene Expression Data

Gene expression data was analyzed using Principal Component Analysis (PCA) to identify patterns and visualize differences between ER⁺ and ER⁻ samples. The dataset was first standardized using `StandardScaler` to normalize feature values. PCA was then applied to reduce the high-dimensional gene expression data into principal components, and the first principal component (PC1) was used for visualization.

Samples were plotted based on their PC1 projection, with different colors representing ER⁺ and ER⁻ classes. This analysis helped in observing clustering patterns and the separation of samples based on their gene expression profiles.
